## Stack utilisée

- Material UI

## Test technique
### => Pré-requis

- Le chargement des données de la prochaine promo
-	Un formulaire d’inscription

La page affichera le nom de la promo, la date de début et la date de fin de la formation et le formulaire d’inscription.

### => Contenu du fichier JSON

- Nom de la promo
-	Date de début de la formation
-	Date de fin de la formation

Charger les informations de ce JSON en en-tête de votre page.

### => Contenu du formulaire

- Firstname (input - free)
- Lastname (input - free)
- E-mail (input - required)
- Password (input - required)
- Password confirmation (input - required)
- Status (select "teacher, teacher assistant, student" - free)
- Subscribe to newsletter (checkbox - free)
- I have read terms and conditions (checkbox - required)
- Submit (button - clickable when form is valid)

### => Stack

- Utiliser un framework JS (React, Vue ou Angular), de préférence React
-	Pour le styling, utiliser un framework CSS (ou une librairie UI du framework JS)

### => Critères d'évaluation principaux

-	Respect de la consigne
-	Code propre et optimisé

### => Critères d'évaluation secondaires
-	Une validation visuelle et pour chaque champs
-	Aspect visuel de la page
-	Simplicité de lancement (avec un README)
